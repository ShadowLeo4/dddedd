# TooManyProxies (Tomp)

## Design

- Both HTTP and HTTPS will be supported with the exception of invalid https websites (expired, self signed cert).


## Paths

| Path       | Description |
| ---------- | ----------- |
| /tomp/html | HTML pages. |
| /tomp/css  | CSS pages.  |
| /tomp/js   | JS pages.   |
